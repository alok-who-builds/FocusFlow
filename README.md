# FocusFlow

FocusFlow is a cinematic productivity web app built to make studying immersive, animated, and satisfying.

## ✨ Features

- ⏳ Fully customizable Pomodoro timer
- 🎵 Ambient study sounds
- 🌌 Animated cinematic UI
- 📊 Productivity analytics
- ✅ Task management system
- 🔥 XP & streak tracking
- 🖥️ Fullscreen focus mode
- ⚡ Smooth transitions & effects
- 💾 localStorage support

## 🛠️ Built With

- HTML
- CSS
- JavaScript
- GSAP
- Chart.js
